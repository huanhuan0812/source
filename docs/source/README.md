# 这是我的知识仓库

### 注意
- **对于普通markdown文件，在文件开头添加：**
```markdown
---
title: "自定义标题"  # 显示在侧边栏的文本
order: 2            # 排序位置
---

# 正文内容...
```
- **对于index.md/README.md添加：**,
```markdown
---
title: "章节名称"    # 显示在侧边栏的文本
order: 1            # 排序位置
collapsed: false     # 默认不折叠
---

# 章节内容...
```
